Second Wind
A multiplayer real time strategy game in which you control a floating island in search for resources on the map. Resources are finite and so you must compete against other players by engaging in a chess-style turn based battle. The resources you win can be used to upgrade your island and create units for the army you use in battles. Each unit has unique abilities and so choosing the right ones to develop play styles and strategies is crucial to winning a battle. See my GitHub for links.
Naughts and Crosses
A naughts and crosses web game that you can play between two devices in real-time using ReactJS, NodeJS and Websockets. It uses the device's location to find games nearest to you so you know which ones to connect to. felixxwu-xo.herokuapp.com

Music Generator
A C# Windows app that can randomly generate 16 bars of "House" music, complete with drums, main melody and a chord sequence.
